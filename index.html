What are Loops?

Each day, we wake up in the morning, take a bath, brush our teeth, eat breakfast, travel to the office or school, do our office work or attend classes, go home, eat lunch and dinner, and sleep. Everyday, we do the same things over and over again. A lot of things occur in cycles, and one of them is our daily habits. In programming, we call these loops. Loops permit a set of instructions to be repeated untila condition is reached. In Java, there are three loop structures.

Foor Loop
for (initialization; test condition; increment/operation) { statement/s }

The for loop has three main parts separated by three semicolons:

  • initialization - initial values of variables that will be used in the loop.
  • test condition - a boolean expression that should be satisfied for the loop to continue executing the statements within the loop's scope.
  • increment/operation - dictates the change in value of the loop control variable every time the loop is repeated

for loops are commonly used when the number of iterations needed is already known.

Example: public class For_Loop { public static void main(String[] args) { for(int Ctr=1;Ctr<=5;Ctr++){ System.out.println(Ctr); } } }
While Loop

The while loop is another interesting loop structure in java. Here the condition is at the beginning and the set of instructions inside the loop will be repeated until this condition becomes false. Unlike a for loop, the condition does not necessarily involve a loop control variable

public class While_Loop { public static void main(String[] args) { int Ctr=1; while(Ctr<=5) { System.out.println(Ctr); Ctr++; } } }

The output of While_Loop class is the same as the For_Loop class. This shows that there is an alternative way to produce the same output with another loop structure. Unlike the for loop structure, the variable Ctr was declared and initialized outide of the while loop.

The variable Ctr was incremented on line 9. If it was not incremented, its value will remain 1, causing the program to run endlessly. This is called infinite loop.

while(int Ctr<=5) { System.out.println(Ctr); Ctr++ }

You cannot declare and/or initialize Ctr within the parentheses of the beginning of a while loop because the syntax of a while loop only allows conditional statenments to beincluded within it.

while (Ctr<=5) { int Ctr; System.out.println(Ctr); Ctr++; }
Do-While Loop

The do-while loop is the only loop which executes at least once. This is due to the fact that the conditon for the loop is located at the last part.

do { --statement/s-- } while (boolean condition); public class Do_While { public static void main(String[] args) { int Ctr=1; do { System.out.println(Ctr); Ctr++; }whie (Ctr<=5); } }

The output of the Do_While class is the same as the two previous programs. As previously noted in the While_Loop class, the declaration and initialization parts are outside the loop to avoid errors.

Nested Loops
public class Nested_For { public static void main(String[] args) { for(int Row=1;Row<=10;Row++) { for(int Column=1;Column<=10;Column++) { System.out.print(Row*Column+"\t"); } System.out.println(); } } }

The Nested_For class prints the multiplication table. To do this, it needs two loops, where one loop is inside the other. This way it is called a nested loop

What are Exceptions?

If you look up the definition of the word "exception" in the dictionary, you will find out that it means something beyond established rules or regulations. In java, there are also exceptions: scenarios or events that might cause your program to produce errors(i.e., when something happens that you didn't expect).

Error Handling

"Java is dynamic. Java makes it possible to reuse previously written programs to lessen the workload of a programmer."

Classes that help handle errors are included with the standard classes that come with the installation of Java. These are called exception classes.

Try and Cath Statements

When using the try and catch statements, it should be remembered that:

  • a try statement should be paired with at least one catch statement.
  • a try statement can have several catch statements after it but the catch statements should have different exceptions to "catch," anticipate, or handle.
  • a catch statement should not be declared without a corresponding try statement.
  • a try statement should be declared before declaring a catch statement.
  • a catch statement should immediately follow a try statement block.
Summary

Loops permit a set of instructions to be repeated until a condition is reached. In Java, there are three loop structures: the for loop, while loop, and do-while loop. The for loop is commonly used when the number of iterations is already known. The while loop is a loop that checks the condition supplied by the user before entering the loop. The do-while loop does the instructions within the loop before checking if the condition supplied by the user is satisfied. Loops within loops are called nested loops.

Exceptions in Java are scenarios or events that might cause your program to produce errors. To inform Java that there is a possibility to encounter errors within a block, the code block is placed within the try statement. If an exception is encountered within the scope of the try statement, Java will execute the code within the appropriate catch statement thatis assigned to handle the particular exception. A single try statement can have several catch statements paired with it. A try or catch statement can be optionally followed by a finally statement. The code within the finally statement is executed after atry or catch statement regardless of whether Java encountered an exception or not.
